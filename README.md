# vue3-bms
本项目是基于Vue3及相关技术栈开发的，前后端分离的电商后台管理系统。
主要技术栈：Vue 3.0、Vite 2.0、VueRouter 4.0、Element-plus、Axios、Echarts、Wangeditor
